# The Strategic Mentor

A simple static landing page offering personalized guidance for B.Tech students, built with HTML and CSS.

## Live Preview
Open `index.html` in your browser to view the site.

## Files
- `index.html` — the page markup with sections: hero, stats, about, features, and footer.
- `style.css` — styling, responsive layout, and theme variables.

## Features
- Responsive hero with call-to-action buttons linking to a guidance form and LinkedIn.
- Statistics section showcasing mentorship reach.
- About section with personal story and mission.
- Feature cards highlighting services and network.

## How to run
1. Ensure both `index.html` and `style.css` are in the same folder.
2. Double-click `index.html` or open it in your browser.

## Customize
- Update the hero text, links, and contact phone in `index.html`.
- Adjust colors and fonts in `style.css` (:root variables).
