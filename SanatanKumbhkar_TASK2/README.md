# Personal Portfolio Website

A clean and modern personal portfolio website built with plain HTML and CSS.

## Overview
This project is a single-page portfolio that showcases:
- a hero section with introduction and contact highlights
- an About Me section
- a Skills section
- a Projects section
- an Education section
- a Contact section

## Features
- Responsive layout for desktop and mobile
- Simple and professional styling
- No JavaScript required
- Easy to customize with your own content and image

## Files
- index.html - main page structure
- style.css - all visual styling
- IMG_20250126_132903.jpg - profile image used in the hero section

## How to View
Open index.html in any web browser to view the portfolio.

## Tech Stack
- HTML5
- CSS3
