# Temperature Converter

This project is Task 3 of the OIBSIP web development internship. It is a simple temperature conversion web application that lets users convert temperatures between Celsius, Fahrenheit, and Kelvin.

## Project Structure
- `index.html` - Main HTML structure for the converter UI
- `style.css` - Styling for the page layout and responsive design
- `script.js` - JavaScript logic for temperature conversions and input handling

## Features
- Convert temperature between Celsius, Fahrenheit, and Kelvin
- Instant results on button click or input change
- Clean and responsive user interface

## How to Run
1. Open `index.html` in your browser.
2. Enter a temperature value.
3. Select the source unit and target unit.
4. Click the Convert button to see the result.

## Technologies Used
- HTML
- CSS
- JavaScript
